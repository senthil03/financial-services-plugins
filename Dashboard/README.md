# 📊 Antigravity Financial: Portfolio Crisis Dashboard

This is your premium, web-based "Mission Control" for monitoring the real-money eToro portfolio and simulating the **April 7 US-Iran Strike Deadline**.

## 🚀 Quick Start

1.  **Start the Server**:
    Open a terminal in this directory and run:
    ```bash
    python -m http.server 8000
    ```

2.  **Access the Dashboard**:
    Open your browser and navigate to:
    **[http://localhost:8000](http://localhost:8000)**

## 🛡️ Key Features

-   **Tactical Countdown**: Real-time timer synced to the **April 7, 8:00 P.M. ET** deadline.
*   **Crisis Mode**: Toggle the switch to simulate a **-20% shock** to all high-beta assets (IONQ, BETA, ACHR, etc.).
-   **Live Data**: Visualizes the latest holdings from `portfolio_result.json`.
